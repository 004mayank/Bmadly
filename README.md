# Bmadly

> Run BMAD workflows in your browser- no setup, no CLI, just click and go.

Bmadly is a browser-first execution platform for BMAD.  
It removes the complexity of local setup and makes BMAD accessible to anyone through a simple UI.

---

## Why Bmadly?

BMAD is powerful, but:

- Requires CLI knowledge
- Needs local environment setup
- Hard to onboard new users

**Bmadly changes that.**

→ No installs  
→ No configuration  
→ No terminal  

Just open, run, and get results.

---

## What is Bmadly?

Bmadly is a **cloud execution layer + UI** for BMAD workflows.

It:
- Runs BMAD inside secure containers
- Streams logs in real-time
- Returns outputs instantly in the browser

---

## Core Features (v1)

- ✅ One-click BMAD execution  
- ✅ Real-time log streaming (terminal-like UI)  
- ✅ Containerized runs (secure & isolated)  
- ✅ Instant output access
